Memory Game Time!

Rules

  1. Clicking on a block shows a letter or a number on the block.
  2. Only two blocks can show letters or numbers at a given time.
  3. If the two blocks with the same letters appear they remain visible for the remainder of the game.
  4. If the text on the two blocks are different they disappear in 3 seconds.
  5. Any clicks during this time period (3 seconds from the clicking of the first block) is ignored.
  6. The game is over when all identical pairs are matched.
  7. Try to match all the blocks with least steps.
  8. Have fun!
